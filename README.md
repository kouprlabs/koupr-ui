# Koupr UI

Koupr Design System 🪐

## Getting Started

```sh
pnpm install @koupr/ui
```

## Development

Install [Node.js 18.x](https://nodejs.org).

Enable [pnpm](https://pnpm.io):

```shell
corepack enable
```

### Build and Run

Install dependencies:

```sh
pnpm install
```

Build:

```sh
pnpm run build
```

Watch:

```sh
pnpm run watch
```
