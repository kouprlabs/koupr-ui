# Koupr UI

## Getting Started

```sh
npm install @koupr/ui
```

## Development

Install [Node.js](https://nodejs.org/en/download), preferrably with [NVM](https://github.com/nvm-sh/nvm).

### Build and Run

Install dependencies:

```sh
npm install
```

Build:

```sh
npm run build
```

Watch:

```sh
npm run watch
```
